# password-analyzer-
A professional Python-based cybersecurity tool that evaluates password strength, calculates entropy, checks breach exposure using the Have I Been Pwned API, and generates structured security reports.
Features
Security Analysis

Rule-based password strength evaluation

Entropy calculation (measures resistance to brute-force attacks)

Breach detection using the Have I Been Pwned password API

Secure Reporting

No plaintext password storage

SHA-256 hashing for report safety

CSV security report export

Summary statistics of analysis results

Professional Tooling

Command-line interface (CLI)

Batch password file scanning
Technologies Used

Python

Requests (API integration)

Argparse (CLI interface)

Logging module

CSV reporting

Cryptographic hashing (SHA-1, SHA-25

Secure logging system

Strong password generator
